# Simple Java Console Calculator
A basic Java console-based calculator that performs **addition, subtraction, multiplication, and division**.
This program demonstrates the use of methods, conditional statements, loops, and basic user input handling in Java.
## Features
- Addition, subtraction, multiplication, and division of two numbers.
- Division by zero handling (returns an error message instead of crashing).
- Interactive console menu for choosing operations.
- Continuous calculation until the user chooses to exit.
# Usage
Run the program.
Select an operation from the menu:
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)
5. Exit
Enter two numbers when prompted.
View the result.
Repeat as needed or select "Exit" to terminate the program.
